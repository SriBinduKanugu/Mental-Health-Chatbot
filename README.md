# 🧠 Mental Health Chatbot

An AI-powered Mental Health Chatbot built using Python and Streamlit that provides supportive conversations and a simple user-friendly interface for interaction.

## 🚀 Features

- 💬 Real-time chatbot interaction
- 🧠 AI-generated responses
- 🎨 Simple and interactive Streamlit interface
- ⚡ Fast and lightweight application
- 🔒 Secure API handling using `.env`

## 🛠️ Tech Stack

- Python
- Streamlit
- OpenAI API / LLM API
- dotenv
